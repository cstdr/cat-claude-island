# 本地化待办事项

## 当前状态

菜单栏 (NotchMenuView) 已完成本地化，使用 `String(localized:)` 和 `LanguageManager.shared.localized()`。

---

## 问题：当前实现有两个问题

### 1. 性能问题
`String(localized:)` 不会响应运行时语言切换，只在视图创建时读取一次。

### 2. 维护性问题
两种写法混用，不一致：
```swift
String(localized: "Back")           // 系统标准
LanguageManager.shared.localized("Back")  // 自定义
```

---

## 解决方案：使用 String 扩展

### 新增文件：String+Localization.swift

```swift
// String+Localization.swift
import Foundation

extension String {
    /// 使用 LanguageManager 进行本地化
    var localized: String {
        LanguageManager.shared.localized(self)
    }

    /// 带参数的本地化
    func localized(with arguments: CVarArg...) -> String {
        LanguageManager.shared.localized(self, arguments)
    }
}
```

### 使用方式对比

```swift
// 之前
LanguageManager.shared.localized("Back")
String(localized: "Back")

// 之后
"Back".localized
"Waiting: %@".localized(with: desc)
```
